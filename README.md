# temperature_Converter

The Temperature Converter is a simple web application built using HTML, CSS, and JavaScript. Its primary function is to convert temperatures between Celsius, Fahrenheit, and Kelvin. This project aims to provide an easy-to-use and intuitive interface for users to perform temperature conversions.

# Features

Convert Celsius to Fahrenheit: Enter a temperature value in Celsius, and the application will instantly display its equivalent Fahrenheit value.
Convert Celsius to Kelvin: Similarly, input a temperature in Celsius, and the application will show its corresponding Kelvin value.

# How to Use
Clone or download the project repository to your local machine.
Navigate to the project directory.
Write `cd temperature_converter` in bash/ terminal.
<br>
Open the `index.html` file in your web browser.

Once the web application loads, you'll see a user-friendly interface with input fields and buttons.

## Celsius to Fahrenheit Conversion:

Enter the temperature value in Celsius that you want to convert.
Click the "Convert to Fahrenheit" button.
The equivalent Fahrenheit value will be displayed on the screen.

## Celsius to Kelvin Conversion:

Enter the temperature value in Celsius that you want to convert.
Click the "Convert to Kelvin" button.
The equivalent Kelvin value will be shown on the screen.
Feel free to try different temperature values and perform multiple conversions as needed.
